- [ ] 火力発電機（物燃やす -> 質量Enが熱Enに変換する -> 水を蒸発 -> タービン回す）  
- [ ] 熱電発電機 （熱いも物と冷たい物を用意 -> 熱が移動する -> その際に一部の熱Enが電気Enに変換される）  
- [ ] 可燃型ガス発電機（ガス燃やす -> 質量Enが熱Enに変換する -> 水を蒸発 -> タービン回す）  
- [ ] 不燃型ガス発電機（ガス温める -> 体積増える ->  タービン回す）  
- [ ] 沸騰水型核分裂炉とタービン（²³⁵Uと²³⁸Uを1:19の割合で臨界状態を生み出す -> 分裂の際に一部の質量Enが熱Enに変換する -> 水を蒸発 -> タービン回す）  
- [ ] 加圧水型核分裂炉とタービン（²³⁵Uと²³⁸Uを1:19の割合で臨界状態を生み出す -> 分裂の際に一部の質量Enが熱Enに変換する -> 高圧の水の温める -> その熱で軽水を水蒸気化 -> タービン回す）  
- [ ] 核融合炉とタービン（勉強予定）  
- [ ] 惑星発電機（コア付近まで地下掘る -> ガス温める -> 体積増える ->  タービン回す）  
- [ ] 型恒星発電機（太陽の大気圏に機械送る -> ガス温める -> 体積増える ->  タービン回す）
